# @turtlexyz/sdk

TypeScript SDK for the Turtle Earn v2 API, autogenerated from the OpenAPI spec

## Install

```bash
bun add @turtlexyz/sdk
# or
npm install @turtlexyz/sdk
```

## Usage

```ts
import { getChains, getProtocols, getTokens } from "@turtlexyz/sdk";

// Reads TURTLE_API_KEY and TURTLE_BASE_URL from process.env.
const chains = await getChains();
```

### Environment variables

| Variable          | Default                   | Description                  |
|-------------------|---------------------------|------------------------------|
| `TURTLE_API_KEY`  | _(empty)_                 | Required for authenticated calls |
| `TURTLE_BASE_URL` | `https://earn.turtle.xyz` | API base URL                 |

## Development

```bash
# Regenerate the client from specs/openapi.v2.json.
bun run codegen

# Build (codegen + tsup).
bun run build

# Type-check.
bun run type-check
```

The generated code (`src/client/`) is gitignored and recreated on every build.
The source of truth is `specs/openapi.v2.json` (inside this package), kept in
sync with `turtle-partners` by the `openapi-sync-sdk.yml` workflow.
